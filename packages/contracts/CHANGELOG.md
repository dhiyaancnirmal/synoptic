# @synoptic/contracts Changelog

## Versioning Policy
- Any breaking interface/schema change requires:
  1. Semver major bump for `@synoptic/contracts`
  2. Changelog entry describing breakage
  3. Migration note for downstream consumers (API, MCP, CLI, dashboard)
